# Día 3: Manipulación de Strings.

En desarrollo web, el manejo de texto es constante: limpiar lo que mete un usuario en un formulario, formatear URLs, parsear respuestas de una API o buscar palabras clave. En JavaScript, los strings son inmutables (no se pueden modificar directamente, cada operación devuelve un string nuevo), por lo que entender cómo trocearlos y transformarlos es clave.
📘 Tus herramientas para hoy:

    .trim(): Elimina los espacios en blanco al principio y al final.

    .toLowerCase() / .toUpperCase(): Cambia las mayúsculas/minúsculas.

    .slice(inicio, fin): Extrae una sección de un string.

    .split(separador): Divide un string en un array de substrings.

    .replace() / .replaceAll(): Reemplaza texto.
