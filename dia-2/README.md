# Día 2: Métodos esenciales de Arrays (map, filter, reduce).

Hoy dejamos atrás los bucles for manuales. El objetivo es aprender a usar estos tres métodos nativos que te permiten transformar, filtrar y acumular datos en una sola línea, que es exactamente lo que se busca en el desarrollo moderno con React.

## 📘 La regla de oro del Día 2

    .filter(): Devuelve un nuevo array con una cantidad de elementos igual o menor al original. Se usa para descartar cosas.

    .map(): Devuelve un nuevo array con la misma cantidad de elementos que el original, pero con los datos transformados.

    .reduce(): Toma un array y lo reduce a un solo valor (puede ser un número, un string, un objeto o un array acumulado).
